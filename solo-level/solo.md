**Maqsad:** PHP kodining asosiy sintaksisi va strukturasi bilan tanishish.

* **Nazariy Qism**

  1. PHP fayli qanday boshlanadi: `<?php … ?>`
  2. Ekranga chiqarish:

     ```php
     echo "Hello World!";
     print "Salom Dunyo!";
     ```
  3. Izohlar:

     ```php
     // Bu bitta qatorli izoh
     /* Bu 
        bir nechta

     ```

     ```php

     $ism = "Jinwoo";

     ```
